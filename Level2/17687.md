# [3차] 방금그곡


## Solution

```python
def convert(num, n):
    chart = '0123456789ABCDEF'
    q, r = divmod(num, n)

    if q == 0:
        return chart[r]
    else:
        return convert(q, n) + chart[r]


def get_string(n, t, m):
    result = ''

    for number in range(t * m):
        result += convert(number, n)
        number += 1

    return result[:t * m]


def solution(n, t, m, p):
    str_list = get_string(n, t, m)
    answer = ''
    
    for idx in range(p - 1, t * m, m):
        answer += str_list[idx]
    
    return answer
```
